How TruthOS Works

1. Claim Intake

User submits a claim (text, link, or document). The system classifies the claim type and extracts entities and timelines.

2. Evidence Discovery

Sources are discovered across regions and classified into evidence tiers.

3. Logical Evaluation

Contradictions, physical constraints, and logical consistency are tested.

4. Verdict Generation

Probability-weighted verdict is assigned using deterministic rules.
